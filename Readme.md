# Parking

이 서비스는 머신러닝을 이용하여 CCTV가 주차장의 자동차를 인식하여 빈자리 수를 실시간으로 계산하고, 해당 주차장의 정보를 어플 사용자에게 미리 제공해준다.

이 서비스는 YOLO, ImageZMQ 프로토콜, Android Studio, FireBase 기반으로 구현되었습니다.
